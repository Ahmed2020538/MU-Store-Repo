You are a senior full-stack AI engineer and product architect.

Your task is to completely replace the current fake overlay-based “Try It On” implementation with a real, production-grade Generative AI Virtual Try-On system comparable in quality and UX to modern fashion e-commerce platforms such as ASOS, Zalando, and Amazon.

This is NOT an AR overlay project.

This is a complete AI-powered image generation pipeline.

The final implementation must feel premium, scalable, production-ready, mobile-first, secure, and visually polished.

Payload:

```json
{
  "model_image": "<base64 or public URL>",
  "garment_image": "<base64 or public URL>",
  "category": "tops" | "bottoms" | "one-pieces",
  "mode": "quality",
  "adjust_hands": true,
  "restore_background": true,
  "restore_clothes": true,
  "garment_photo_type": "auto",
  "num_samples": 1

```

Response:

```json
{
  "id": "prediction_id",
  "status": "starting"
}
```

STEP 2 — POLL STATUS

GET /status/{id}

Poll every 2 seconds.
Maximum timeout: 90 seconds.

Success:

```json
{
  "status": "completed",
  "output": ["generated_image_url"]
}
```

Failure:

```json
{
  "status": "failed",
  "error": "..."
}
```

On completion:

* Display generated image immediately
* Cache result locally in state
* Save result to generation history
