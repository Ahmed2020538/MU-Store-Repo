🛠️ ADMIN CONTROL PANEL (CRITICAL REQUIREMENT)
━━━━━━━━━━━━━━━━━━━━━━━

13. Full Admin Customization:
Provide a powerful admin dashboard that allows FULL control over the Hero Section WITHOUT code changes.

Admin must be able to:

- Edit all text:
  - headlines
  - subtext
  - buttons
- Manage multiple languages content

- Control visuals:
  - change background type (static / animated / video)
  - upload images or videos
  - switch banners per language

- Manage AI features visibility:
  - enable/disable AI Stylist
  - enable/disable Try-On
  - enable/disable Voice Shopping

- Edit CTAs:
  - text
  - links
  - styles
